# Sorting Performance Analyzer (SPA)

## Overview
This project compares the performance of three sorting algorithms implemented from scratch in Python:

- Insertion Sort
- Merge Sort
- Quick Sort

The program generates different types of datasets and measures execution time of each algorithm.

## Features
- Implements sorting without using `sorted()` or `list.sort()`
- Generates datasets:
  - Random
  - Already Sorted
  - Reverse Sorted
- Input sizes:
  - 1000
  - 5000
  - 10000
- Measures execution time fairly using copied arrays
- Displays results in table format

## Files Included
- `spa_sorting.py` → Main Python program
- `report.pdf` → Analysis report
- `output.txt` → Console output with timing results
