# Data Management Mini Toolkit (DMMT)

## Overview
This project implements three important non-linear and advanced data structures in Python:

- Binary Search Tree (BST)
- Graph Traversal (BFS & DFS)
- Hash Table with Separate Chaining

The toolkit demonstrates searching, traversal, deletion, and fast key-value storage.

## Features

### Binary Search Tree
- Insert nodes
- Search nodes
- Delete nodes (leaf, one child, two children)
- Inorder traversal (sorted output)

### Graph Module
- Adjacency List representation
- Breadth First Search (BFS)
- Depth First Search (DFS)

### Hash Table
- Insert key-value pairs
- Retrieve values
- Delete keys
- Collision handling using separate chaining
