# Linear Data Structures Toolkit (LDST)

## Overview

This project is based on Unit–2 Assignment of Data Structures. It includes implementation of linear data structures in Python from scratch.

## Implemented Modules

* Dynamic Array
* Singly Linked List
* Doubly Linked List
* Stack (LIFO)
* Queue (FIFO)
* Balanced Parentheses Checker

## Key Features

* Dynamic resizing in array
* Insert, delete, traverse operations in linked lists
* Stack operations: push, pop, peek
* Queue operations: enqueue, dequeue, front
* Parentheses balance checking using stack

## Time Complexity

* Append: O(1) amortized
* Stack Push/Pop: O(1)
* Queue Enqueue/Dequeue: O(1)
* Linked List Search/Delete: O(n)
